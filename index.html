HTML

HTML (Hypertext Markup Language) é uma linguagem de marcação que define a estrutura e o significado do conteúdo de uma página web. É o bloco de construção mais básico da web e é usado para criar páginas web e aplicações online.

CSS

CSS é a sigla para Cascading Style Sheets, que em português significa Folhas de Estilo em Cascatas. É uma linguagem de programação que define como os dados são apresentados no navegador.
